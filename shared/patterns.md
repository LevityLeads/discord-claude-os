# Common Patterns -- All Teams

Reusable patterns for common tasks. Copy and adapt.

---

## Role Announcement Pattern

When switching between specialist modes, announce:
\`\`\`markdown
:emoji: **Role responding...**

[actual response]
\`\`\`

Emoji + bold role name + "responding..." on its own line.

## Memory Update Pattern

\`\`\`markdown
### [Date] -- [Brief title]

- [What happened]
- [Decision made / outcome]
- [Next steps if any]
\`\`\`

## Escalation Pattern

When handing off to general assistant:
\`\`\`markdown
This needs cross-project context. Escalating to general.

Context: [Brief summary of what's needed and why]
\`\`\`

## Research Summary Pattern

\`\`\`markdown
## Summary
[1-2 sentence TLDR]

## Key Findings
- [Finding 1]
- [Finding 2]

## Recommendations
1. [Action 1]
2. [Action 2]

## Sources
- [Source 1]
\`\`\`

## Error Report Pattern

\`\`\`markdown
**[Role] ran into an issue**

**What failed:** [specific thing]
**What I tried:** [steps taken]
**What I need:** [from user or another role]
\`\`\`

## Handoff Pattern

When passing work to another team or mode:
\`\`\`markdown
Handing this to [Team/Role] -- [brief context]

---

**For [Team/Role]:**
- [Task 1]
- [Task 2]
- [Relevant context]
\`\`\`
